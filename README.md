#Todo-App (React Native)
A simple and elegant Todo application built with React Native. This app allows users to add, edit, search, mark as done, and delete todos. All todos are persisted locally using AsyncStorage.

##Features
Add Todo: Quickly add new tasks to your list.
Edit Todo: Update existing tasks with ease.
Delete Todo: Remove tasks you no longer need.
Mark as Done: Check off completed tasks.
Search: Instantly filter your todos by keyword.
Persistent Storage: Todos are saved locally using AsyncStorage.
Responsive UI: Clean and modern interface with keyboard handling.
